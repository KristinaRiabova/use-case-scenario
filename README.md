## 📌 Product Overview

**Game Tracker and Quest Manager** is an application where users can:
- add games to their library  
- track progress  
- complete quests  

It helps players organize their gaming activity and monitor their achievements in one place.

---

## 🎯 Use Case

**Title:** Complete a Quest  

**Goal:**  
The user marks a quest as completed and updates game progress.
